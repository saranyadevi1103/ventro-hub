# VentroHub

VentroHub is a lifestyle e-commerce website built using HTML, CSS, and JavaScript.

## Features
- Product categories & filters
- Product detail pages
- User authentication
- Cart & checkout
- Blog & about pages

## Folder Structure
- assets → product images
- css → styles
- js → application logic
- data → products data

## How to Run
Open index.html using Live Server.
